# Generátor diplomů

Umožňuje automaticky generovat diplomy ze seznamu jmen.

## Použití

Do souboru `seznam_jmen.txt` uložit na každou novou řádku jméno studenta a pohlaví oddělené středníkem.

- m = muž
- f = žena

Např:

```text
Karel Novák; m  
Karla Nováková; f
```

Poté pusťte soubor `gen_dip.py`.

````commandline
python gen_dip.py
````
